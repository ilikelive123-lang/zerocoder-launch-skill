# Инструкция: как я составляю ТЗ для копирайтера

Яна приносит концепт мероприятия → я создаю ТЗ, по которому копирайтер пишет тексты.

ТЗ — это не готовый текст. Это чёткая инструкция: что писать, с какого угла, для кого, в каком порядке. Копирайтер строит текст сам.

---

## Что мне нужно от Яны перед составлением ТЗ

- Что продаём (продукт, ссылка)
- Даты мероприятия / дедлайны писем
- Сегменты: нейро / технари / оба
- Спикер(ы): имя + регалии (от чьего лица пишем каждое письмо)
- Ключевые практические результаты участника
- Бонусы с названиями (если есть), кодовые слова
- Есть ли ограничение/дедлайн для срочности
- Формат регистрации: с 1 кликом или без
- Что продаём платного в финале (если есть), промокод, ссылки
- Программа/презентация (ссылка, если есть)

## Структура ТЗ — как я её строю

### Блок 1: Общие вводные

```
**ОБЩИЕ ВВОДНЫЕ**

[Дата и время мероприятия]
[Название мероприятия]
[Короткое описание — 1–2 предложения что за эфир, кто ведёт]

ЦА:
- [кто это]
- [кто это]

Бонусы:
- Всем участникам: [название]
- За кодовое слово «[слово]» в чате: [что именно]

Программа:
[ключевые поинты эфира — что узнает участник]

Спикер: [Имя], [должность]. [Регалии 2–3 строки.]

Что продаём в конце: [продукт + ссылка]
Промокод / спецпредложение: [если есть]

Важно: Все тексты адаптировать отдельно под email и бота/тг-канал и при написании
ориентироваться на данные рекомендации:
https://docs.google.com/document/d/1AAYNQxkBrWyDCb4eY-O-n4AzkBVJ_1uBBv4ARs1pHO8/edit?usp=sharing
По этой теме можно чаще добавлять визуал в письма.

Важные поинты по дизайну и верстке — желательно текст письма отдавать так, чтобы он
подходил для подобной верстки:
https://drive.google.com/file/d/1K1MtdinPSylpiMbJzk66G5o2-xt05mtS/view?usp=sharing
Драфт нового письма здесь:
https://docs.google.com/document/d/1ScZJdq8P9hDlF-UVp_hCzFI3B3Ji4icIqtaqD3MT7UM/edit

Всегда помнить эту ЦУ — ИНФА ПО АНОНСАМ:
https://docs.google.com/document/d/1DhYyYdvzk2TrsRCdK4Jlboi4PvlL-S-AeO6-Z53l3Rg/edit?usp=sharing

Правила языка:
— Всё, что можно писать по-русски — пишем только по-русски.
— Всё что можно перевести транслитом (perplexity → перплексити) — пишем транслитом.
— Что не поддаётся переводу — максимум сноска в скобках, или синоним, или убрать.
— Обращение на «ты». Без корпоративщины.
— {first_name} в начале бот-сообщений.
— Без негативного посыла (база отписывается).
— Кнопка/ссылка в первых двух абзацах.
— Конкретные числа хотя бы раз на письмо.

Формат CTA:
[если с 1 кликом] Кнопка: [РЕГИСТРАЦИЯ В 1 КЛИК] + обязательная сноска:
«Мы знаем, что у многих есть проблемы с загрузкой сайтов, поэтому сделали возможность
регистрации буквально в 1 клик. Но если твой интернет стабилен — можно зайти на сайт тут.»
[если без 1 клика] Стандартные кнопки: ЗАРЕГИСТРИРОВАТЬСЯ / ЗАНЯТЬ МЕСТО / ЗАЛЕТАЙ и т.д.

Объём:
— Email: до 400 слов, блочная верстка (каждый смысловой блок — отдельно)
— Telegram-бот: 400–800 символов, максимум 1000, личное обращение с {first_name}
— Telegram-канал: та же структура, но без персонализации

В конце каждого письма обязательно:
РЕКЛАМА ООО "ЗЕРОКОДЕР"
ИНН 9715401631

Перед сдачей: проверить текст на стоп-слова через нейросеть, поменять формулировки.
```

### Блок 3: Список писем с ТЗ на каждое

Для каждой группы писем указываю:
- Тип группы: **Анонсы бесплатника** / **Анонсы платника** / **Ласт кол**
- Дедлайн (дата и время, строго) — если не указан, оставляю пустым
- Номер и краткий посыл в скобках курсивом:
  - от чьего лица: Кирилл / Ксюша / спикер / Зерокодер
  - угол атаки — с какой мысли заходим
  - что обязательно включить (поинты, бонусы, цифры, снятие страха)
- Если есть сегментация — сразу под описанием письма: «Для технарей — ...» / «Для нейроэнтузиастов — ...»
- Если нужна адаптация для канала — отдельной строкой после письма курсивом

**Пример блока писем в ТЗ:**

```
Анонсы бесплатника, пишем на ты

Дедлайн:

1. ТЗ по письму (от лица Кирилла — друзья, вот НОВЫЙ практикум!
Та тема, что набирает актуальность. Проведём с нашим техдиром, разберём весь процесс —
как зарабатывать на ботах от 100 000 руб. в эпоху блокировок. Основные поинты + подарки.
Первый мощный оффер, количество бесплатных мест ограничено)

Важно, нужно собрать два варианта анонсов:
Для технарей — на языке понятном данной аудитории, ориентир — первые два сегмента ЦА
Для нейроэнтузиастов — на простом понятном языке, ориентир — вторые два сегмента ЦА

Дополнительно адаптировать анонс для канала.

2. ТЗ по письму (упор на аудиторию и пользу для каждого сегмента исходя из программы
+ хайповый, притягательный заголовок)

Для технарей — на языке понятном данной аудитории, описываем профит только для них
Для нейроэнтузиастов — на простом языке, описываем профит только для них

Дедлайн:

3. ТЗ по письму (чёткое, структурное — основные поинты из веба + чёткий заголовок
+ ограничительная мера)

Для технарей — упор на невероятные возможности инструмента
Для нейроэнтузиастов — упор на простоту работы с инструментом

4. ТЗ по письму (ограничительная мера — веб уже завтра, не пропускай;
повторяем программу — самые значимые поинты; заголовок об упущенной выгоде)

5. ТЗ по письму (шальной анонс, краткий, структурный — общее, но цепляющее для обоих сегментов)


Анонсы платника

Дедлайн:

Информация о продукте: [продукт + ссылка] [промокод если есть]

1. ТЗ по письму (мини-продажа с краткими выжимками из спецоффера
и напоминанием о повторе в 19:00)

2. ТЗ по письму (задействовать кейсы — хочешь также оптимизировать работу?
+ упомянуть крутой оффер)

3. ТЗ по письму (основные выжимки из презы + подробно про плюшки тарифов + оффер)
```

## Типы посылов — примеры из реальных ТЗ

**Бесплатник / первый анонс:**
- «Ты уже не раз слышал про вайб-кодинг, но ничто не сравнится с возможностью пощупать своими руками»
- «От лица Ксюши: давно ли ты откладывал этот шаг? Вот твой шанс»
- «Время сейчас тревожное — и именно сейчас нужен дополнительный аэродром»

**Упор на результат:**
- «Ты сам соберёшь машину, которая поедет — сайт, генератор постов и автопостинг за 3 дня»
- «3 инструмента на руках: сайт-дайджест, генератор постов, автопостинг»

**Упор на ЦА и для кого это:**
- «Кому зайдёт: если не хочешь менять профессию, но деньги нужны вчера — для простых задач на ИИ достаточно 2 часов в день»

**Закрытие регистрации:**
- «От лица Кирилла — закрываем реги, самые сильные поинты из программы + подарки»

**Продажа платного / нарастающий рынок:**
- «Нарастающий спрос на вайб-кодинг, будь среди первых + упор на тариф ВИП с гарантией»
- «Предприниматели тратят 100к+/мес на фрилансеров. Но с ИИ ты даёшь тот же результат за 20к и в 10 раз быстрее»

**Повышение цен / дедлайн:**
- «Поднимаем цены — заходим через то, что в абонементе стало ещё больше»
- «Ласт кол: уже завтра поднимаем цены, напоминаем про повышение, призываем поторопиться»

**Государство и тренды (кризисный контекст):**
- «В РФ запустили нацпроект "Экономика данных", выделив 2 трлн. руб. на развитие нейросетей»

## Разбивка по сегментам — как формулировать

**Нейро-сегмент (нетехнари):**
- Язык: простой, про быт, работу, автоматизацию рутины
- Акцент: «кодить не нужно», «доступно без опыта», «для любой профессии»
- Продукты-триггеры: Нейросети для жизни, Гигачат, Перплексити, Визуал с ИИ, Доп. доход с ИИ

**Технари/Бизнес:**
- Язык: можно упоминать API, агенты, мультиагентные системы — с пояснениями
- Акцент: «глубина», «управление агентами», «монетизация экспертизы», «стек»
- Продукты-триггеры: Claude Code, Вайб-кодер, Автоматизатор n8n, ИИ для руководителей

## Чеклист перед отдачей ТЗ

- [ ] Посыл каждого письма — угол атаки, а не описание программы
- [ ] Указан контекст: что сейчас болит у аудитории, откуда заходим
- [ ] Для каждого сегмента — свой посыл (если сегментируем)
- [ ] Указано от чьего лица пишем каждое письмо
- [ ] Указан формат регистрации: с 1 кликом или без
- [ ] Есть конкретные данные: что продаём, кто спикер, какие бонусы, какой дедлайн
- [ ] Все три канала: email + бот + канал (или оговорено исключение)
- [ ] Адаптация для канала — указана отдельной строкой где нужно
- [ ] Нет противоречивых требований
- [ ] Если есть готовые варианты писем — дать ссылки как образец

## Что я делаю с концептом от Яны
