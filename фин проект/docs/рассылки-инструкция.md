# Инструкция: как я пишу тексты рассылок

Яна приносит концепт мероприятия → я создаю email, Telegram-бот и Telegram-канал тексты.

---

## Шаг 0: Что анализирую до написания

1. **Тип письма:** анонс бесплатника / продажа платного / напоминание / дедлайн / закрытие регистрации
2. **Сегмент:** нейро (нетехнари) / технари+бизнес / оба → если оба — два отдельных письма
3. **Формат регистрации:** с 1 кликом или без — это меняет структуру CTA и наличие сноски
4. **От чьего лица:** Кирилл / Ксюша / Зерокодер — влияет на тон всего письма
5. **Контекст ЦА прямо сейчас:** что их беспокоит, что актуально (кризис / блокировки / тренды / рынок труда). Эмпатический хук строится из этого, не из шаблона.
6. **Тип заголовка:** хайповый или контентный. Пропорция 4:1 — максимум 1 хайповое на 4 контентных, никогда два хайповых подряд.

---

## Структура каналов

Под каждое письмо делаю **три адаптации**:
- **Email** — полная версия, 7 блоков
- **Telegram-бот** — личное обращение с `{first_name}`, ссылка рано, до 1000 символов
- **Telegram-канал** — без персонализации, чуть другой тон. Оговаривается отдельно если нужна

---

## Email

### Заголовок — всегда 5 вариантов

Для каждого письма даю 5 вариантов темы: минимум 1 хайповый + остальные контентные. Копирайтер или Яна выбирают.

**Хайповые (OR выше, CTR ниже в 80% случаев):**

Псевдо-уведомление — не чаще 1–2 раз в месяц:
- `Оформлена заявка 4567🔥`
- `❗️Регистрация не подтверждена [мероприятие]`
- `Назначена встреча с [эксперт] [Бесплатно]`
- `Отказ по вашему резюме [Готовы рассмотреть, если освоите ИИ]`

ЗАВТРА + конкретика:
- `ЗАВТРА! [название практикума]`
- `💚 Не пропусти! ЗАВТРА [мероприятие] [Беги регистрироваться]`

Число + дедлайн:
- `19 ИИ в 1! Практикум уже [дата] 🔥`
- `❗️[N] курсов по цене 2х только до конца дня!`

**Контентные (CTR стабильнее):**

Задача + снятие барьера:
- `Собери бота, который [результат] [Своими руками]`
- `Научу [действие] за 30 минут [Бесплатно]`

Вопрос-провокация:
- `«Я перестала пользоваться [инструмент]» [дата расскажу почему]`
- `Что будет с [тема] после [триггер]? [Ответим на практикуме!]`

Эксперт + тема:
- `Бесплатное обучение по [тема] от [имя] 🔥`

**Правила заголовка:**
- Длина: 30–40 символов (приоритет) — 60 максимум
- `[дописка в скобках]` — не длиннее 5–7 слов
- Всегда есть глагол действия
- Если иностранный инструмент в заголовке — пишу как есть (повышает OR)

### Тело письма — 7 блоков под блочную верстку

Письмо разбиваю на логические блоки — каждый блок отдельно (под верстку Зерокодера). Не единый текстовый поток.

**1. Персонализация + эмпатический хук**

Начало: `{first_name}, привет, это [Кирилл/Ксюша] 👋`

Хук — про состояние/боль читателя прямо сейчас, НЕ про программу:
> «Время сейчас тревожное. Каждый день — адаптация к новым условиям.»
> «Ты уже наверняка видел кучу видео. Но возникает простой вопрос: а что ДЕЛАТЬ-то с этим всем?»

Хук должен отражать актуальный контекст ЦА этого мероприятия, не шаблон.

**2. Первый CTA** — до основного контента, сразу после хука

*Формат с 1 кликом:*
```
[РЕГИСТРАЦИЯ В 1 КЛИК]

*Мы знаем, что у многих есть проблемы с загрузкой сайтов, поэтому сделали возможность регистрации буквально в 1 клик по кнопке выше! Но если твой интернет стабилен, и ты хочешь подробнее изучить сайт — сделать это можно тут.*
```

*Формат без 1 клика:*
```
[ЗАРЕГИСТРИРОВАТЬСЯ БЕСПЛАТНО] / [ЗАНЯТЬ МЕСТО] / [ХОЧУ НА УРОК]
```
(без сноски)

**3. Польза по жизненным сценариям** — НЕ по программе курса
```
✔ Без технического бэкграунда
✔ Без смены профессии
✔ Даже если есть всего 2 часа в день
```
или:
```
⚡ Для работы: [конкретное действие]
⚡ Для заработка: [конкретное действие]
⚡ Для бизнеса: [конкретное действие]
```

**4. Конкретные поинты из программы** — что узнает/получит участник (с цифрами)
```
✅ [конкретный результат 1]
✅ [конкретный результат 2]
✅ [конкретный результат 3]
```

**5. Бонусы с конкретными названиями и ценностью**
- `📎 Подборка «100 промтов из коробки»`
- `📎 Как получить годовую подписку на Перплексити (стоимость $240) БЕСПЛАТНО`
- Если есть денежная ценность — называю прямо

**6. Снятие страха** — 1–2 строки прямо перед финальным CTA
- `Кодить не нужно!`
- `Доступно в РФ без посредников`
- `Даже если у тебя нет опыта и портфолио`

**7. Финальный CTA** — эмоционально сильнее первого

*С 1 кликом:*
```
[ЗАРЕГИСТРИРУЙСЯ В 1 КЛИК ЗДЕСЬ]
```

*Без 1 клика:*
```
[БЕГИ РЕГИСТРИРОВАТЬСЯ] / [ЗАЛЕТАЙ] / [ЗАБРАТЬ 🎯]
```

**Обязательно в конце письма:**
```
РЕКЛАМА ООО "ЗЕРОКОДЕР"
ИНН 9715401631
```

---

### Что не делаю в теле письма
- Не начинаю с описания программы
- Не пишу биографию эксперта длиннее 1 предложения перед первым CTA
- Не ставлю единственный CTA в самом конце
- Не пишу «полезные материалы» — только конкретные названия бонусов
- Не забываю про цифру: хотя бы одна на письмо («за 30 минут», «80 000 за проект», «$240 бесплатно»)
- Не применяю КАПС и жирный везде подряд

---

### Важно про язык
- Обращение на «ты»
- При первом упоминании нового инструмента — пояснение в скобках: `Перплексити (ИИ-поисковик, заменяет Гугл)`. Дальше — просто «Перплексити».
- Всё что можно написать по-русски — пишу по-русски. Транслит: Перплексити, Гигачат, Яндекс ДжиПиТи. Оставляю английским: Claude Code, n8n, VK API, Python — только если нет русского аналога и пояснение уже дано.
- Никакого негативного посыла — не «у тебя всё плохо», а «вот что ты можешь»
- Конкретные числа: «за 30 минут», «выгода 90%», «от 80 000 за проект»

## Telegram-бот

**Личное обращение с персонализацией.**
Средний размер: 400–800 символов. Максимум — 1000.

### Структура

**Строки 1–3 — про состояние читателя, не про событие** (это единственное что видно в превью):
- ✅ `{first_name}, привет, это Кирилл 👋 Время сейчас тревожное...`
- ❌ `13 мая состоится практикум по OpenClaw`

**Ссылка рано** — после 1–2 абзаца, не только в конце

*С 1 кликом:*
```
[Зарегистрируйся в 1 клик] → ссылка

*Мы знаем, что у многих есть проблемы с загрузкой сайтов — сделали регистрацию в 1 клик. Если интернет стабилен — можно зайти на сайт тут.*
```

**Навигация по болям или поинты:**
```
✔ Без технического бэкграунда
✔ Без смены профессии
✔ Даже если есть 2 часа в день
```
или:
```
🔹 Если хочешь [боль 1] → на эфире [решение]
🔹 Если устал от [боль 2] → покажем как
```

**Финал — личный вопрос или образ закрытия:**
- `Ты с нами? → ссылка`
- `❗ Осторожно, двери закрываются 🚪 → ссылка`
- `Лови последние места → ссылка`

**Обязательно в конце:**
```
РЕКЛАМА ООО "ЗЕРОКОДЕР"
ИНН 9715401631
```

## Telegram-канал

**Без персонализации, публичный формат.** Указывается отдельно в ТЗ как "адаптировать для канала".

Отличия от бота:
- Нет `{first_name}`, нет личного «привет, это Кирилл»
- Может быть от имени команды: «Расскажем», «Покажем»
- Тон чуть менее личный, но такой же живой
- Структура та же: хук → ссылка → поинты → финал → реклама

## 10 горячих крючков (вшивать в любой тип письма)

1. **«Завтра»** — самый сильный дедлайн
2. **Денежная ценность бонуса** — «подписка за $240 бесплатно» > «подарок»
3. **«Кодить не нужно»** — главный страх аудитории, снимать перед финальной кнопкой
4. **«Доступно в РФ»** — особенно для зарубежных инструментов
5. **Историческая параллель** — «как в 1991 году интернет казался диковинкой»
6. **Конкретный кейс студента** — «Тамара, организатор праздников, берёт заказы за $500»
7. **«Бесплатно»** — эффективнее в теле, чем в заголовке
8. **«Ограничение»** — «бесплатных мест осталось мало» эффективнее в теле, чем в заголовке
9. **«Двери закрываются»** > «осталось мало мест» (второе — заезженное клише)
10. **Личный голос** — от «Ксюши» или «Кирилла» кликают лучше, чем от «Зерокодера»

## Что работает лучше всего в Telegram

- Геймификация (колесо фортуны, гарантированный приз) — лучший TG-триггер
- Каталог мероприятий «Последний залп» — охватывает все сегменты одним постом
- Личный вопрос «Ты с нами?» — втягивает в диалог
- Цена + выгода в одной строке: `30+ курсов за 3100₽ [выгода 90%]`
- Конкретный кейс из жизни студента — работает лучше абстрактных обещаний
