## Вопросы перед стартом

Задавать всё одним блоком. Пропускать то, что уже есть в концепте.

1. **Спикер** — кто ведёт?
2. **Дизайн лендинга** — цвета, стиль, настроение, референс-сайт
3. **Сегменты рассылки** — НЕЙРО + Технари/бизнес / один сегмент / вся база без деления?
4. **Сколько основных анонсов бесплатника?**
5. **Нужно ли 6-е шальное письмо?** (не делится на сегменты, фиксированный текст)
6. **Платные анонсы** — нужны? Если да:
   - Какой продукт + **ссылка на курс/программу**
   - **Сколько платных писем** (стандарт — 3)
   - Промокод, скидка
   - Бонусы за регу с веба
   - Нейроабонемент — да/нет?

Тип мероприятия (эфир или акция) — определять самостоятельно по концепту.

## Алгоритм выполнения

После получения ответов — сначала создать Google Drive папку, затем выполнять субагенты по порядку:

### 0. Создать Google Drive папку
```bash
python3 scripts/create_drive_folder.py \
  --name "[название мероприятия] — [дата]" \
  --save-to письма-для-воронки/Зерокодер/[название]/
```
Получить `FOLDER_ID` из вывода скрипта. Передавать во все три скрипта ниже через `--folder-id`.

### 1. Воронка (`voronki-dlya-zapuskov`)
- Алгоритм: из `.claude/skills/voronki-dlya-zapuskov/skill.md`
- Создать бот + email файлы в `письма-для-воронки/Зерокодер/[название]/`
- Запустить скрипты с `--folder-id FOLDER_ID`, получить две ссылки

### 2. Лендинг (`tz-dlya-dizov`)
- Алгоритм: из `.claude/skills/tz-dlya-dizov/skill.md`
- Создать файл в `фин проект/Сайты для бесплатника/[название]/`
- Запустить скрипт с `--folder-id FOLDER_ID`, получить ссылку

### 3. ТЗ для копира (`tz-dlya-kopirov`)
- Алгоритм: из `.claude/skills/tz-dlya-kopirov/skill.md`
- Создать файл в `фин проект/тз для копиров/[название]/`
- Запустить скрипт с `--folder-id FOLDER_ID`, получить ссылку
